# RemoteManager

**RemoteManager** 是一款轻量级远程管理工具，支持管理多台服务器的远程连接，数据存储在本地 JSON 文件中。

---

## 功能特性

- 添加、编辑、删除服务器信息
- 支持用户名 / 密码登录远程桌面
- 双击服务器行直接连接
- 右侧按钮支持手动连接
- 查看和编辑本地 JSON 数据文件 (`servers.json`)
- 自动刷新列表，编辑 JSON 后可立即生效


---

## json示例

```json
[
  {
    "Id": "b37e5cf4-b9b5-4b0d-90a8-cffb7795bd47",
    "Name": "60测试服",
    "Address": "110.9.161.60",
    "UserName": "administrator",
    "Password": "a#*(aa"
  }
]

```
